# Fraud Detection using Graph Neural Networks (GNN)

This project detects fraudulent transactions using Graph Neural Networks.

## 🧠 Idea
Transactions are modeled as a graph:
- Nodes → Users
- Edges → Transactions

The model learns patterns to classify fraud.

## 🚀 Features
- Graph-based fraud detection
- GCN model using PyTorch Geometric
- Ablation study (topology impact)

## 📦 Tech Stack
- Python
- PyTorch
- PyTorch Geometric
- Pandas
